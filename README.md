# Trade Insight PWA

A beautiful, local-first Progressive Web Application designed for traders to log their entries, exits, profit/loss, and save their chart screenshots.

## Features

- **Progressive Web App**: Install it on your phone or desktop and use it offline.
- **Local Storage**: All trades and chart images are stored locally in your browser (IndexedDB). No sign-up required, total privacy!
- **Dynamic Dashboard**: Calculates your Win Rate, total PnL, and number of trades dynamically.
- **Image Support**: Seamlessly attach screenshot proofs or chart drawings (like from TradingView) for later review.
- **Dark Mode Aesthetic**: A modern dark-theme design perfect for trading setups. 

## Get Started

Install dependencies:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Build the application for production:
```bash
npm run build
```
